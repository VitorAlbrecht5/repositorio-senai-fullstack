## 1. O que sao operadores em programacao?

**Vitor:** São simbolos para realizar alguma ação, por exemplo: ||, &&, e !.

## 2. Para que servem os operadores aritmeticos? Cite exemplos.

**Vitor:** Servem para realizar calculos matematicos, como por exemplo o +, /, %, * e -.

## 3. Qual a diferenca entre / e % em JavaScript?

**Vitor:** A diferença é que a / é a divisão, ja a % é o resto da divisão.

## 4. Como descobrir se um numero e par usando o operador %?

**Vitor:** Basta fazer o numero % 2 == 0, se true é par.

## 5. O que e uma expressao em programacao?

**Vitor:** É um trecho que calcula algo e da um resultado.

## 6. O que significa precedencia de operadores?

**Vitor:** É a ordem correta que os operadores são executados, como na matemática.

## 7. Por que os parenteses podem mudar o resultado de uma expressao?

**Vitor:** Porque o que esta dentro dos parenteses é executado primeiro.

## 8. Qual a diferenca entre =, == e ===?

**Vitor:** 

= -> usado para atribuir valor

== -> quando o valor é igual 

=== -> quando é estritamente igual

## 9. Por que e recomendado usar === em JavaScript?

**Vitor:** Por segurança, pois garante que o valor e o tipo sejam iguais.

## 10. O que os operadores relacionais retornam: texto, numero ou boolean?

**Vitor:** Boolean

## 11. Quando usamos o operador >? E o operador >=?

**Vitor:** 

Quando precisa que o valor informado seja maior do que o informado, se usa: >

Quando quer que o valor informado seja maior ou igual, se usa: >=

## 12. Quando usamos o operador < ? E o operador <=?

**Vitor:** 

Quando precisa que o valor informado seja menor do que o informado, se usa: <

Quando quer que o valor informado seja menor ou igual, se usa: <=

## 13. Para que serve o operador logico &&?

**Vitor:** Serve para retornar true se ambas as condições forem verdadeiras.

## 14. Para que serve o operador logico ||?

**Vitor:** Serve para retornar true se pelo menos uma das condições forem verdadeiras.

## 15. Para que serve o operador logico !?

**Vitor:** Inverte o valor da condição, se for true -> vira false, e vice-versa

## 16. Qual a diferenca entre uma condicao simples e uma condicao composta?

**Vitor:**

Simples -> Executa a ação apenas se for verdadeiro
Composta -> Oferece 2 caminhos, se for verdadeiro e se for falso

## 17. Explique uma situacao real em que duas condicoes precisam ser verdadeiras ao mesmo tempo.

**Vitor:** Uma situação real seria por exemplo no caso do embarque de voo, é necessário ter a passagem e o documento da pessoa.

## 18. Explique uma situacao real em que apenas uma entre duas condicoes precisa ser verdadeira.

**Vitor:** Na hora de pagar algo, ela pode pagar em dinheiro ou cartão.

## 19. Como o console.log() pode ajudar a encontrar erros no codigo?

**Vitor:** Ele ajuda pois exibe no terminal, e se tiver algum erro ele vai mostrar lá.

## 20. Por que devemos testar o mesmo programa com valores diferentes?

**Vitor:** Para garantir que ele funcione em diferentes situações.
