## 1. O que é programação?
**Vitor:** A programação é escrever instruções para o computador. Serve para criar aplicativos e site por exemplo.

## 2. O que é um algoritmo?
**Vitor:** O algoritmo é uma sequência de passo finita.

## 3. Qual a diferença entre algoritmo e linguagem de programação?
**Vitor:** O algoritmo é uma sequencia de passos. E a linguagem de programação é usada para escrever o código.

## 4. O que é JavaScript?
**Vitor:** Uma linguagem de programação.

## 5. Onde o JavaScript pode ser utilizado atualmente?
**Vitor:** Pode ser usado para desenvolvimento de sites, apps e outros sistemas.

## 6. O que é o Node.js?
**Vitor:** É um ambiente de execução.

## 7. Qual é a função do Node.js no desenvolvimento com JavaScript?
**Vitor:** A função dele é rodar o JavaSript no computador, fora da web.

## 8. O que é o VS Code?
**Cibely:** O VS Code é uma IDE.

## 9. Qual a diferença entre um editor de texto comum e um editor de código?
**Cibely:** A diferença entre eles é que, enquanto editores de texto comuns são voltados para a criação de documentos e redação sem formatação complexa, os editores de código são ferramentas especializadas para programadores.

## 10. O que é o terminal?
**Cibely:** Terminal é a interface onde você digita comandos baseados em texto para instruir o computador.

## 11. Para que serve o comando node arquivo.js?
**Cibely:** Para executar um arquivo. 

## 12. O que é uma variável?
**Cibely:** Uma variável é um espaço na memória do computador.

## 13. Qual a diferença entre let e const?
**Cibely:** Variáveis declaradas com let podem ter seus valores alterados posteriormente, enquanto as declaradas com const são constantes e não podem receber um novo valor após a inicialização.

## 14. O que são tipos de dados? Cite exemplos.
**Cibely:** Classificações que indicam ao computador a natureza, o tamanho e quais operações podem ser realizadas sobre uma determinada informação. Exemplos: String, number e Boolean.

## 15. Qual a principal diferença entre JavaScript e TypeScript?
**Cibely:** O JavaScript é dinamicamente tipado, enquanto o TypeScript adiciona uma camada de tipagem estática.
